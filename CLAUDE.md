# FENIX IA - Landing Page WEB1

## Proyecto
Landing page para **FENIX IA 360**, plataforma B2B SaaS de analítica de comunicaciones con IA.
Esta es la **variante WEB1** (tema oscuro, storytelling emocional).

## Estructura
- `index.html` - Página completa (HTML + CSS inline + JS inline)
- `logo.png` - Logo de FENIX IA
- `CNAME` - Dominio: web1.fenixia360.com

## Stack Técnico
- HTML5, CSS3 (inline), Vanilla JavaScript
- Google Fonts: Inter + JetBrains Mono
- Sin frameworks, sin build tools, sin dependencias
- Deploy: GitHub Pages

## Diseño WEB1
- **Tema**: Oscuro (#09090b) con acentos púrpura (#7c3aed)
- **Enfoque**: Storytelling emocional - "¿Estás escuchando?"
- **Audiencia**: Decisores de negocio que conectan emocionalmente
- **Animaciones**: Orbes flotantes, gradientes, pulse effects

## Producto FENIX IA
- **FENIX Pulse**: Analítica IA (transcripción, sentimiento, oportunidades) - €79/mes
- **FENIX Connect**: Hub de comunicaciones unificado - €49/mes
- **FENIX 360**: Bundle completo (Pulse + Connect) - €99/mes
- **Canales**: Llamadas, Email, WhatsApp, Telegram, Meta
- **Sectores**: Clínicas, Inmobiliarias, E-commerce, Asesorías, Seguros

## Variantes Hermanas
- WEB2 (web2.fenixia360.com): Tema claro, enfoque ROI, teal (#0d9488)
- WEB3 (web3.fenixia360.com): Tema claro, enfoque técnico, azul (#4d65ff)

## Reglas de Estilo
- Todo el contenido en **español**
- Mantener consistencia de precios y features entre las 3 variantes
- CSS usa variables custom (--color-primary, etc.)
- Responsive: breakpoints en 900px y 600px
- Mantener contraste de texto suficiente sobre fondos oscuros
- No añadir dependencias externas innecesarias
